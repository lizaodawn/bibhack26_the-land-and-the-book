### Overview

This repository contains the curated dataset and code developed by Team w@nder for the [KU Leuven Bibliotech Hackathon 2026](https://www.arts.kuleuven.be/hackathon/).
Our project maps and visualizes The Land and the Book by William McClure Thomson, integrating text, images, and routes to trace his 19th‑century travels through the Holy Land alongside present‑day landscapes.

### Contents
**/curated dataset/** – curated Excel/CSV datasets with scraped and cleaned text

**/data processing codes/** – scripts for data processing and map generation
